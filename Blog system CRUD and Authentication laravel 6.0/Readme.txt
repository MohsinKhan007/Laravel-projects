Lasapp:
	 A simple Blog posts app in which a user can add a blog and can edit or remove it. The Blog contains title, body and image of each blog post.
	 The app has a login and register system and in-build laravel authentication and security system. An user can only edit or delete his/her posts. 



1. Folder lasapp contains the main files of the of the app.
2. lasapp.sql is the mysql datase of the app.

How to Run:
1.copy the lasapp folder in you xampp/htdocs folder.
2.Create the database lasapp and export the database file in it.

Note: your database username should be root and password must be 123456 you can change it in phpmyadmin portal.(app has also these settings).
